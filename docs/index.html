Agentic RAG

Cypher 2025 Hackathon — Dynamic Knowledge Assistant

Project Overview

Agentic RAG is a modular multi-agent knowledge assistant designed to dynamically adapt to different query types. It integrates local knowledge sources, live web search, and reasoning agents to provide reliable, verifiable, and context-aware answers.

Quickstart

git clone https://github.com/CaptnSalazar/agentic-rag-cypher2025.git
cd agentic-rag-cypher2025
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run_demo.py --query "What causes sudden server crashes?"

Key Features

  • Multi-agent pipeline (Router, Retriever, Reasoner, Verifier)
  • Dynamic query handling (Factual, Open-ended, Reasoning-heavy)
  • Evidence-grounded responses with provenance and citations
  • Minimalist and clear architecture for real-world applicability
